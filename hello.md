# hello git

## git 명령어 요약

- clone : 원격 저장소 복사
- add   : 스테이지 영역에 작업 파일 추가
- commit: 세이브, 스테이지 영역의 파일들을 가지고 커밋을
            만들 수 있다.
- push  : 원격 저장소에 커밋을 업로드 한다.


## 명령 되돌리기.
 - 아무내용이나 적어서 저장 후 
 - 소스트리 상에서 '코드뭉치 버리기' 하고 다시 돌아오면
 - 거짓말처럼 작업하던 파일도 원상복귀 되어있다.
 - 명령어로는 checkout 이 있다.
 - 신비롭고 놀라운 git의 세계 !
