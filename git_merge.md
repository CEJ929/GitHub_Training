# git branch병합

## 병합이란?
 - 하나의 브랜치를 현재 브랜치와 합치는 것.
 - 현재 브랜치느 HEAD 브랜치라고 한다.
 - 예를 들어 HEAD 브랜치가 main 이고 version2 브랜치를
   병합하면 version2 의 내용이 main에 반영되게 된다.


## 병합하기 1
 - 헤드 브랜치에 변경사항이 없고
 - 병합 대상 브랜치가 헤드로부터 시작된 경우 아주 쉽게
   병합이 가능하다.
